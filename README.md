# Pingpong League

탁구 리그/토너먼트 운영을 위한 웹 서비스입니다.

## 주요 기능

- 리그 방 생성/관리 (단식·복식·단체전)
- 운영 방식: 조별 풀리그 / 예선+본선 토너먼트
- 토너먼트 표기: 단방향/양방향 브라켓
- 풀리그 순위 기준 선택: 승수/승점
- 참가자 관리: 본인 참가, 방장 강제 추가, 게스트 추가, 일괄 추가
- 단체전 팀 구성: 자동 배정(랜덤) + 수동 편집/저장
- 경기 결과 입력/수정 및 자동 완료
- 최종 순위/결과 요약

## 운영 프로세스 (기본 흐름)

1. 리그 방 생성  
2. 참가자 모집  
3. (단체전) 팀 구성 확정  
4. 리그 시작 → 조별 경기 자동 생성  
5. 예선+본선 모드일 경우 본선 진출자 확정 후 토너먼트 진행  
6. 리그 종료 및 최종 결과 확인  

## 화면/모듈 구조

- 리그 방
  - 생성/수정/상세/참가자 관리
  - 단체전 팀 구성 UI (자동/수동)
- 대진표
  - 조별 풀리그 경기표 + 순위표
  - 본선 토너먼트 브라켓
- 결과
  - 최종 순위 및 요약

## 기술 스택

- Backend: Spring Boot, Spring MVC, Spring Security, JPA(Hibernate)
- Frontend: Thymeleaf + Vanilla JS + jQuery 브라켓
- DB: H2 (로컬)

## 실행 방법

```bash
./gradlew bootRun
```
